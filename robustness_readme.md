DESCRIPTION/REPRODUCIBILITY: - This dataset contains rasters of
landscape robustness for the Bern Landscape Change Explorer project. -
All rasters are projected in the CH1903+ / LV95 coordinate reference
system and share the same extent and resolution (1 hectare). - For
detailed descriptions of the data generation process, raster value
interpretation and suggested colour scheme, please refer to the
accompanying metadata files: - BeschreibungMetadatenRobustheit_DE.pdf -
BeschreibungMetadatenRobustheit_ENG.pdf

Two raster-based indicators are provided for each topic for the
robustness evaluation: performance and stability. The performance
indicator is stored in files with the prefix Mean_sum_of_change\_ and
describes the mean cumulative change value across all scenarios and
future points in time. The stability indicator is stored in files with
the prefix Undesirable_deviation_sum_of_change\_ and describes the
extent of undesirable negative deviations across all scenarios. Both
indicators are available for biodiversity (BD), ecosystem services (ES),
and the combination of biodiversity and ecosystem services (BD-ES). The
robustness classes on the platform are derived from the joint
classification of these two indicator grids and are therefore not
included as a separate grid layer.
