DESCRIPTION/REPRODUCIBILITY: - This dataset contains rasters of land use
and land cover (LULC) change scenarios for the Bern Landscape Change
Explorer project. - All rasters are projected in the CH1903+ / LV95
coordinate reference system and share the same extent and resolution (1
hectare). - For detailed descriptions of the data generation process,
raster value interpretation and suggested colour scheme, please refer to
the accompanying metadata files: - BeschreibungMetadatenLULC_DE.pdf -
BeschreibungMetadatenLULC_ENG.pdf

The simulations were carried out in five-year increments for the period
2025 to 2060. Three climate scenarios, five economic scenarios, three
population scenarios, and five land use scenarios were taken into
account. The land use scenarios control the future transition
probabilities between land use classes. A total of 2,025 LULC files are
available for all combinations of year, climate, economy, population,
and land use scenario. The file names follow the pattern:
lulc-`<year>`{=html}-`<climate scenario>`{=html}-`<economic scenario>`{=html}-`<population scenario>`{=html}-`<land use scenario>`{=html}-
canton.tif Example: lulc-2025-rcp45-ref_peri_urban-low-ei_cul-canton.tif
